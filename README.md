🧠 Quiz App – Logic Building Project
📌 Project Overview

This is a Quiz Application built using JavaScript to strengthen logic-building and problem-solving skills. The project focuses on handling user interactions, managing application state, and implementing core programming concepts like conditions and event handling.

🚀 Features
Multiple-choice questions
Answer selection and validation
Dynamic question rendering
Score tracking and final result display
Quiz restart functionality
Smooth navigation between questions

⚙️ Core Functionalities
Answer Selection:
Implemented using event listeners to capture user input and validate answers
Question Handling:
Dynamically loads questions one by one using JavaScript and DOM manipulation
Score Calculation & Display:
Tracks user score using variables and displays final result after quiz completion
Restart Quiz:
Resets score, question index, and UI to restart the quiz from the beginning

🛠️ Technologies Used
HTML
CSS
JavaScript

🎯 Learning Outcomes
Improved logic building using JavaScript
Gained hands-on experience with DOM manipulation
Learned event handling and user interaction
Understood state management (score, question index)
Practiced writing clean and structured code.
